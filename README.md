## 🛠️ Getting Started
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/moolmin/commit-mate.git
   ```
2. Install YARM packages
   ```sh
   yarn install
   ```
3. Run the app
   ```sh
   yarn dev
   ```
